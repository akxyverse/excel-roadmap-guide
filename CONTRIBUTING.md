# Contributing to The Excel Roadmap

Thanks for wanting to help improve this! Contributions of all sizes are welcome.

## Ways to contribute

- 🐛 **Report a mistake** — spotted a wrong function name, broken link, or typo? Open an issue.
- 💡 **Suggest a topic** — think a skill is missing from a level? Open an issue describing it.
- 🔗 **Add resources** — good free tutorials, datasets, or practice files for any level.
- 🎨 **Improve the design** — the roadmap images, or the README itself.

## How to submit a change

1. Fork this repository.
2. Create a new branch for your change: `git checkout -b fix/short-description`
3. Make your change.
4. Commit with a clear message: `git commit -m "Fix: correct XLOOKUP description in Level 3"`
5. Push your branch and open a Pull Request.

## Guidelines

- Keep the tone consistent — practical, direct, no filler.
- If adding a resource, make sure it's free or has a solid free tier.
- Small, focused pull requests are easier to review than large ones.
